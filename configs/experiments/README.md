# Experiment Configs

Named `ExperimentConfig` instances for recurring or reference experiments.
Each file is a JSON document that validates against `schemas/experiment_config.schema.json`.

Used for: baseline re-runs, regression checks, documented reference configurations.
Ad-hoc experiments generated by the Search Strategist Agent are not stored here —
they live in the metrics database. Only configs worth naming and keeping go here.

Added in Phase 1 as the Mode A sweep is built out.
